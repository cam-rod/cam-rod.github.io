If you need to send me a message privately, you can encrypt it using this form. Download my GPG public key from envs.net, and then upload the file below or paste the contents into the public key box. Then, enter your message into the second box, and click encrypt. You can copy the generated message and send it over email or otherwise.

This page uses HTML5 and a local copy of OpenPGP.js 5.7.0 to perform the encryption. You can view the minified Javascript of OpenPGP.js , and also the script running this page.
